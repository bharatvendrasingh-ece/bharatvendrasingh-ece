## 🚀 Featured Projects

### 🏠 Smart Home Automation using IoT
Wi-Fi-based home automation system that controls appliances remotely and monitors temperature & humidity in real time on a Blynk mobile dashboard — cutting manual effort by ~80%.

| Layer | Technology |
|---|---|
| Microcontroller | ESP8266, Arduino |
| Connectivity / Dashboard | Blynk |
| Actuation | Relay Module |
| Sensing | DHT11 |
